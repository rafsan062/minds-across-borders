About This Project

"Minds Across Borders" is an interactive visual analytics dashboard exploring the global mental health crisis. It compares countries across treatment gaps, psychiatric resources, and socioeconomic factors using coordinated views:

  • Choropleth Map — Color-coded country comparison with multiple metrics
  • Scatter Plot — Explore relationships between variables with size-encoded bubbles
  • Bar Rankings — Sortable illness and preparedness rankings by region
